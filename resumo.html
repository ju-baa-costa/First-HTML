Estrutura do arquivo

Um arquivo HTML é a estrutura do site, portanto, se deve criar um plano, levando em conta as partes mais externas para internas desse site, e então dividindo cada grande seção em partes menores e menores, e as principais tags utilizadas para isso são header(cabeçalho), main(parte principal do site) e footer(rodapé). Existem especificidades para cada uma dessas, mas isso será abordado no tópico Tags

Tags

Existem inúmeras tags no html, sendo as principais, div, header, footer, main, article, p, h1, h2... E cada uma delas tem sua particularidade, por exemplo, pode haver apenas uma main por página, a tag div toma todo o espaço horizontal da página, e muito mais!
Aqui vai o link do site da W3schools , onde você pode ler e estudar mais sobre as tags do html.

Atributos

Para facilitar a organização, estilização e indexação do seu site, existem diversos atributos que podem ser adicionados em cada tag, como por exemplo, o id e class. Além disso, tags específicas precisam de atributos específicos para funcionar, como a tag "a", que adiciona um link, apenas quando esse é inserido no atributo "href", ou a tag "img" que apenas adiciona a imagem ao site, quando o arquivo escolhido está inserido no atributo "src".
Algo importante de lembrar é que com o HTML5, alguns atributos, principalmente de estilização, não tem uso recomendado, uma vez que o papel de gerenciar a parte estética do site, recaí totalmente sobre o CSS.

Midia

Diversos tipos de arquivo podem ser adicionados em seu site, como:

  • Imagens, com a tag "img";
  • Áudios, com a tag "audio";
  • Vídeos, com a tag "video";
  • Outras páginas web, com a tag "iframe";

Todas essas tags dependem do atributo scr para funcionarem, e nele, é colocado o caminho para, ou o link da, mídia escolhida
Além disso, é necessário ter atenção quanto a acessibilidade do seu site, utilizando atributos como:

  1. Descrições de imagens;
  2. Legendas em vídeos;
  3. Tag figure, que agrupa todos os elementos referentes a uma imagem, como legenda, descrição e outros, em apenas um bloco semanticamente, o ue permite que leitores de tela possam passar essas iformações a seus usuários de forma mais precisa, assim melhorando a acessibilidade de seu site!
Tabelas no HTML

As tabelas, funcionam da mesma forma que no excel, sendo formadas por linhas e colunas, e tendo seu encontro chamado de células. No html, a tag "table" a que inicia uma tabela, sendo seguida das tags "tr" (table row/linha), que podem conter tanto as células de cabeçalho "th" ou de dados comuns "td". A quantidade de elementos th/td, definem quantas colunas a tabela terá, enquanto os elementos tr, definem a quantidade de linhas.
A estilização das tabelas pode ser feita por meio do CSS, com adição de bordas, cores, elementos interativos e muito mais.
É também importante se atentar à acessibilidade das tabelas, que pode ser melhorada por meio da tag "summary".

Formulários

Formulários são ferramentas especialmente úteis para coletar dados dos usuários, e são amplamente utilizados pela web. Para criá-los, existem tags específicas: "form", que cria o formulário e "input" que cria um campo de entrada de dados. A tag input tem o atribito "type" no qual se seleciona o tipo de dado que quer ser recebido, dentre eles temos:

  • number, podendo também er atributos de mínimo e máximo aceito;
  • text, que recebe textos pequenos;
  • textarea, sendo possível adicionar textos maiores;
  • range, que cria uma barra com um controle deslizante sobre ela, permitindo que o usuário escolha um valor dentre seu intervalo;
  • color, disponibiliza um seletor de cor ao usuário;
  • email, requer que um email válido seja enviado;
  • url, da mesma forma que o email, exige um link válido;
  • date, que permite que seja selecionada uma data;
  • checkbox, adiciona elementos com caixas seletoras, podendo ser chekcados nenhum, um, ou mais deles;
  • radio, cria elementos diversos, mas apenas um pode ser selecionado pelo usuário;
  • file, permite que o usuário envie arquivos de diversos tipos;
  • button, normalmente usado para fazer o envio do formulário, ou resetá-lo;

Para que os dados sejam manipulados, é possível utilizar javascript, e para estilizar o formulário, o CSS é a ferramenta certa!
